Hardware Compatibility ​

Here is the hardware list confirmed after joint testing by the community and can be used for installation reference.

Unknown：The hardware's compatibility is unknown and requires you testing

Compatible：The hardware is compatible with LoongArch platforms and compatible with 'old-world' and 'new-world' distributions

Partial Support：The hardware can be compatible with LoongArch platforms but some features are not supported, and may be compatible with both 'old-world' and 'new-world' or only 'old-world' distributions.

New-world Available：The hardware is compatible with LoongArch platforms, but is only available on 'new-world' distributions

Unsupported：This hardware is incompatible with LoongArch platforms and is incompatible with any 'world' distributions

Notice ​

  1. Memory compatibility is tested with the latest firmware version by default and can only be used as a reference, some motherboards may not support ECC or ECC REG, so you need to ask the manufacturer if you want to use UDIMM/ECC or RDIMM memory.
  2. POWEV's brands include: GLOWAY, Asgard, CUSO, and SINKER, all of which are categorized under POWEV in the database, with different brands noted in the `Arch or Series'.
  3. Partial data sources: 龙芯3A6000内存（硬盘、网卡）兼容性汇总 (Tencent Docs)、UOS Ecology、Loongson Ecosystem. The latter two of these data may be inaccurate and require further testing.
